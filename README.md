## Installation

1. Clone this repository:  

```
git clone https://github.com/JoelenCruz/Cub3d_level4_42 cub3d_jcruz
```

2. Navigate to the project directory:  
```
cd cub3d_jcruz
```

3. Compile the project using the Makefile:
```
make
```
